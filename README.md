<h1>あとがき</h1>
<p>初めてのITプロジェクトでした
<p>ITの経験が全然ない状態だったから辛かったんです。
<p>今もわからない部分が沢山あるけど、どの部分をわからないかも、わからない状態で苦しいんです。
<p>多分、何ヶ月ぐらいは今の状態が繋がるんだと思います。
<p>プロジェクトをおしまいする時は、納得出来ないエラーとかは出なくてよかったでした。
<p>狙った部分を考えて綺麗に作動するのを作るのは面白いでした。
<p>出来ない部分も結構あったけど勉強する時間も苦しいけど楽しい時間でした。
